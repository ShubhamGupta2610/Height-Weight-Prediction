🚀 QUICK START
──────────────

1️⃣ Install Dependencies:
   $ pip install -r requirements.txt

2️⃣ Place Dataset:
   Copy height-weight.csv to data/ folder

3️⃣ Choose Your App:

   OPTION A - Streamlit (Recommended - Easiest):
   $ python run_streamlit.py
   → Opens http://localhost:8501

   OPTION B - Flask (Professional):
   $ python run_flask.py
   → Opens http://localhost:5000

   OPTION C - Jupyter Notebook:
   $ jupyter notebook
   → Open notebooks/3.0-Simple+Linear+Regression.ipynb

   OPTION D - Docker:
   $ docker-compose up streamlit
   → Opens http://localhost:8501

🐳 DOCKER SUPPORT
─────────────────

Build:
  $ docker build -t height-weight-predictor .

Run Streamlit:
  $ docker run -p 8501:8501 height-weight-predictor

Run Flask:
  $ docker run -p 5000:5000 height-weight-predictor \
    python flask_app/app.py

With Docker Compose:
  $ docker-compose up streamlit    # Streamlit
  $ docker-compose up flask        # Flask
  $ docker-compose up              # All services
